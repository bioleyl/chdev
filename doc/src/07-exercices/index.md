# Exercices

Bienvenue dans la section exercices. Ces exercices sont liés aux chapitres de formation — chaque exercice correspond aux concepts vus dans les pages de documentation.

## Liste des exercices

| Section | Thème | Page de cours |
|---------|-------|---------------|
| **[TypeScript](./typescript.md)** | Typage, interfaces, génériques, `this` | TypeScript |
| **[Vue — Fondamentaux](./vue-basiques.md)** | Templates, directives, computed, watchers, cycle de vie | Introduction au frontend |
| **[Communication entre composants](./communication-composants.md)** | Props, évents, v-model, defineModel | Communication entre composants |

## Comment utiliser les exercices

1. **Lisez** le chapitre de formation correspondant
2. **Essayez** l'exercice sans regarder la solution
3. **Vérifiez** votre réponse en ouvrant le `<details>`
4. **Testez** dans votre projet CHDev avec `npm run dev:web`

> 💡 **Conseil** : Copiez le code des solutions dans des fichiers `*.vue` dans `frontend/src/components/` pour les tester en direct.
